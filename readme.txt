An interactive clothing designer that supports Adobe Illustrator templates and allows users to add SVG designs and edit text elements.

== Description ==

Clothing Designer is a powerful and intuitive tool that enables your customers to customize clothing items with their own designs. The plugin provides a user-friendly interface for uploading SVG files, adding text, and customizing designs from different views of garments.

### Key Features

* **Multiple View Support**: View and customize the front, back, left, and right sides of clothing
* **Adobe Illustrator Integration**: Use Adobe Illustrator (.ai) files as clothing templates
* **SVG Support**: Upload and manipulate SVG files
* **Text Editing**: Add and edit text, including text within SVG files
* **Layer Management**: Control the stacking order of design elements
* **Responsive Design**: Works on both desktop and mobile devices
* **Elementor Integration**: Easily embed using shortcodes in Elementor and other page builders
* **WooCommerce Compatible**: Can be integrated with WooCommerce product customization

### Perfect For

* Custom T-shirt shops
* Print-on-demand services
* Apparel designers
* Promotional product companies
* Custom merchandise stores

### Developer-Friendly

The plugin is built with a modular architecture making it easy to extend and customize for your specific needs. It uses industry-standard libraries like Fabric.js for canvas manipulation.

== Installation ==

1. Upload the `clothing-designer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Clothing Designer' in the admin menu to add templates
4. Use the shortcode `[clothing_designer template_id="1"]` to embed the designer on any page

== Frequently Asked Questions ==

= Can I use AI files as templates? =

Yes, the plugin supports Adobe Illustrator (.ai) files as templates. The plugin will attempt to convert AI files to SVG format for use in the designer. For best results, we recommend saving your AI files as SVG files directly from Adobe Illustrator.

= Can users edit text in SVG files? =

Yes, the plugin can extract text elements from SVG files and allow users to edit them directly in the designer interface.

= Does it work with Elementor? =

Yes, you can easily embed the designer in Elementor pages using the shortcode widget.

= Can I customize the appearance of the designer? =

Yes, the plugin comes with a clean, modern design that works well with most themes. You can also customize the CSS to match your brand.

= Is it responsive? =

Yes, the designer interface is fully responsive and works well on both desktop and mobile devices.

= What file types can users upload? =

Users can upload SVG, PNG, and JPEG files to add to their designs.

= How do I create different views of clothing items? =

Each view (front, back, left, right) requires its own template. Create separate templates for each view and link them together in the admin interface.

== Custom Templates ==

For best results with custom templates:

1. Create your template in Adobe Illustrator or another vector graphics program
2. Save each view (front, back, left, right) as separate SVG files
3. Make sure text elements you want to be editable are actual text elements in the SVG, not outlines
4. Upload your templates in the Clothing Designer admin interface
5. Use the provided shortcode to embed the designer on your pages
